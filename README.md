# SPOTIFY APP
This website is a spotify app.

## FEATURES
* It consist of user authentication
* Data is fetched through spotify api.


## TECHNOLOGIES
* REACTJS
* Material UI
* Spotify Api
* Context Api
* CSS
* Host on Netlify
